# Notes book

**Проєкт:** Застосунок для ведення переліку справ з можливістю додавання, видалення та збереження завдань.

**Функціонал:**  
- Додавання завдань з назвою та описом  
- Видалення завдань через кнопку "Delete"  
- Збереження завдань у localStorage  
- Завантаження списку завдань після перезавантаження сторінки  
- Перемикання теми (світла/темна) з збереженням вибору в localStorage  

**Технології:**  
JavaScript (ES6 Modules), localStorage, модульна структура коду, DOM-маніпуляції, CSS (темна/світла тема)
