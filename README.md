# Internet Shop

**Project:** An online store that allows users to browse products, manage a Wishlist and Cart, search items, filter by categories, and view product details in a modal window.

**Features:**  
- Display of product categories and products from the API with pagination (12 items per page)  
- Product search by keyword  
- Product details preview in a modal window  
- Add/remove products to/from Wishlist and Cart with live counter updates  
- Wishlist and Cart pages rendered using Promise.all for fetching products by ID  
- Display of total items count and total price in the Cart  
- Loader shown during data fetching  
- ScrollUp button and light/dark theme toggle with persistence in localStorage  
- User notifications powered by iziToast  

**Technologies:**  
JavaScript (ES6 Modules, async/await), Fetch API / Axios, Promise.all, localStorage, DOM manipulation, CSS (light/dark theme), iziToast
